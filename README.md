# coursecycle-app

Front-end for [Coursecycle][0].

## Requirements

* NodeJS
* gulp (`npm install -g gulp`)
* webpack (`npm install -g webpack`)
* tsc (`npm install -g typescript`)

## Instructions

1. `npm install`
2. `gulp`
3. Fire up a browser at http://localhost:8080

[0]: https://coursecycle.com
